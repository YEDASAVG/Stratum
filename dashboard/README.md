# Stratum Dashboard

**Layers of Insight** - Self-hosted intelligent log analysis with AI chat.

## Features

- **Overview** - Real-time stats and metrics
- **Logs** - Browse and filter logs with live tail
- **Chat** - Ask questions about your logs using AI
- **Anomalies** - Automatic anomaly detection

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser.

## Tech Stack

- Next.js 15 + React 19
- Tailwind CSS + shadcn/ui
- Poppins font
